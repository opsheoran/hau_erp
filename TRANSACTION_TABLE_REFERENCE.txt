TRANSACTION TABLE REFERENCE (VERIFIED FEB 2026)

This file records verified master/transaction tables for Establishment -> Transactions pages.
Only add entries here once the table has been confirmed against the live database schema.

1) SAR/ACR Admin Transaction (establishment.sar_admin_transaction)
- Primary Transaction Table: SAR_Employee_Mst
  - PK: pk_sarid
  - Links:
    - fk_empid -> SAL_Employee_Mst.pk_empid (employee name/code, department/designation)
    - fk_finid -> SAL_Financial_Year.pk_finid (Financial Year / "Session" filter values like CO-11, ES-10)
    - fk_categoryid -> category mapping used by UI (T/N/F -> 1/2/3)
  - Status fields used for tabs:
    - IsSubmit (char) : submitted flag
    - FinalApproval (bit) : final approval flag

- Session/Year Master Table used by dropdown:
  - SAL_Financial_Year (pk_finid)
  - Display can be derived from date1/date2 year range (e.g., 2020-2021)

- Department Master Table used by dropdown:
  - Department_Mst (pk_deptid, description)

Notes:
- The previously referenced table SMS_Session_Mst does NOT exist in the current database (confirmed by SELECT failure).
- SAR_Employee_Mst.fk_finid values match SAL_Financial_Year.pk_finid.
